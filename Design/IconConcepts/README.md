# WorkoutApp Icon Concepts

These concepts are not wired into `AppIcon.appiconset` yet.

Files:
- `01-orbit-progress.svg` - timer ring + progress bars
- `02-rack-stack.svg` - rack / barbell / stacked-set metaphor
- `03-motion-plates.svg` - motion curve + plate nodes
- `04-monogram-wave.svg` - brand-led W symbol
- `index.html` - local comparison gallery

Suggested direction:
- If you want the icon to reflect the current product UX most closely, start with `01-orbit-progress.svg`.
- If you want the most gym-specific silhouette, start with `02-rack-stack.svg`.
- If you want a more branded icon that can scale beyond "just workouts", start with `04-monogram-wave.svg`.
